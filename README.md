## Setup

1. Clone the repo
2. Run `npm install`
3. Copy `.env.example` to `.env` and add your Gemini API key (or add it via Settings in-app)
4. Run `npm start` to launch the Electron app

## Tech Stack
- HTML / CSS / Vanilla JavaScript (no frameworks)
- Bootstrap + Yeti theme
- Node.js + Express
- SQLite (better-sqlite3)
- Electron
- Google Gemini API
